Demo 4: REST API ja virhekäsittelijä

Tämä on demo REST API -palvelinsovelluksesta, jossa virheiden käsittely on toteutettu erillisen virhekäsittelijämiddlewaren avulla. Sovelluksen toimintaperiaate on sama kuin demossa 3, mutta virhetilanteiden käsittely on keskitetty yhteen paikkaan erilliseen virhekäsittelijään.

Testaa demon toiminnallisuutta Postman-sovelluksella. Luo Postman-sovelluksella GET-, POST-, PUT- ja DELETE-pyyntöjä url-osoitteeseen http://localhost:3004/api/ajopaivakirja testataksesi REST API:n toimintaa.
